🔧 TROUBLESHOOTING
================================================================================

"Connection refused" error:
  → Make sure Ollama is running (ollama serve)

"Port 5000 in use" error:
  → Change port in AI.py (line 235) to 5001

"Module not found" error:
  → Run: pip install --upgrade pip
  → Run: pip install -r requirements.txt

Need more help?
  → Check QUICKSTART.md (Troubleshooting section)

🌐 API ENDPOINTS
================================================================================

1. POST /api/chat
   Search for products
   {"message": "Find a laptop under $1000", "budget": 1000, "preferences": "laptop"}

2. POST /api/set-alert
   Set price drop alert
   {"product_id": 1, "target_price": 1200, "email": "you@example.com"}

3. POST /api/track-price
   Track product price
   {"product_id": 1, "product_name": "Dell XPS", "current_price": 1299}

4. GET /api/get-price-history
   Get price history
   ?product_id=1

See API_DOCS.md for detailed examples in cURL, Python, and JavaScript.
